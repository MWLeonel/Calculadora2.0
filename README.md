<h1>Projeto: Calculadora</h1>

<h2>Objetivo</h2><br>
<h3>Criar um software que permitam que o usuário efetue operações matemáticas (o usuário digita os números desejados, e o programa exibe o resultado):

- Soma
- Subtração
- Multiplicação
- Divisão
- Exponenciação
- Radiciação

</h3>
